# TMSBOX - Transportation Management System
## By Reversecube

### Overview
TMSBOX is a comprehensive Transportation Management System for Odoo 18 that provides complete fleet and logistics management capabilities.

### Features
- **Waybill Management**: Track shipments from origin to destination
- **Travel Planning**: Manage complete journeys with multiple waybills
- **Route Management**: Define routes with waypoints and distance tracking
- **Transport Units**: Manage trucks, trailers, and containers
- **Expense Tracking**: Monitor fuel, tolls, maintenance, and other costs
- **Advance Payments**: Track driver advances and reconciliation
- **Financial Analysis**: Real-time profit/loss calculations
- **Multi-company Support**: Works with Odoo's multi-company feature
- **Full Integration**: Seamlessly integrates with Fleet, Contacts, Accounting, and Sales

### Installation
1. Copy the `tmsbox_reversecube` folder to your Odoo addons directory
2. Update the apps list: Settings → Apps → Update Apps List
3. Search for "TMSBOX" and click Install
4. Configure your routes, transport units, and start managing your operations

### Configuration
After installation:
1. Go to TMSBOX → Configuration → Routes to define your routes
2. Add transport units under Configuration → Transport Units
3. Ensure drivers are marked in Contacts (is_driver checkbox)
4. Start creating waybills and travels

### Usage
1. Create a Travel for a driver and vehicle
2. Add Waybills to the travel with cargo details
3. Track expenses during the journey
4. Complete the travel and view profit/loss analysis
