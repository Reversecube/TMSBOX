Quick Start Guide

1. Initial Configuration

Start by setting up your basic configuration. Go to TMSBOX ‚Üí Configuration ‚Üí Routes to define your frequently used routes with waypoints and distances. Then add your transport units under Configuration ‚Üí Transport Units.

2. Driver Setup

Ensure your drivers are properly configured in the Contacts module. Mark them as drivers by checking the appropriate field. This allows them to be selected in travels and waybills.

3. Creating Travels

Navigate to TMSBOX ‚Üí Operations ‚Üí Travels and create a new travel. Select the driver, vehicle, route, and record the starting odometer reading. Confirm the travel to begin operations.

4. Managing Waybills

Create waybills for each shipment within a travel. Add cargo details, customer information, origin, destination, and pricing. Track the waybill through draft, confirmed, in transit, and delivered states.

5. Tracking Expenses

Record all expenses related to the travel including fuel, tolls, maintenance, and meals. Link expenses to travels or specific waybills for accurate cost allocation.

6. Financial Analysis

View real-time profit/loss analysis on each travel. The system automatically calculates total revenue from waybills, total expenses, and net profit, helping you make informed business decisions.

Module Structure

The TMSBOX module consists of the following components:

Models

  • tms.waybill: Shipment documents with cargo details and pricing
  • tms.travel: Complete journeys with multiple waybills
  • tms.route: Predefined routes with waypoints
  • tms.unit: Transport units (trucks, trailers, containers)
  • tms.expense: Travel expenses with categorization
  • tms.advance: Driver advance payments and reconciliation

Views

  • Form views with statusbar workflows and smart buttons
  • Tree views with conditional formatting and summations
  • Kanban views for visual management
  • Advanced search views with filters and grouping
  • Chatter integration for communication and tracking
